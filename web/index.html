Q&A

What is a “tempdecal.wad”?

Tempdecal.wad is a spray format for GoldSrc engine games such as Half-Life, Counter-Strike 1.6, and Sven Co-op. Only preset sprays are available by default. By replacing a tempdecal.wad, you can use any images for sprays, even a full color image!

Where do I have to place a tempdecal.wad?

Here's a summary of folders. Save your tempdecal.wad into the specified folder.

Sven Co-op
[Game Folder]\svencoop
Counter-Strike 1.6
[Game Folder]\cstrike
Half-Life
[Game Folder]\valve

To open [Game Folder], go to Steam Library, right-click your game, then select Manage > Browse local files.

Besides, you should set the tempdecal.wad Read-only to keep it from being overwritten by the game. To set it Read-only, right-click the file, open Properties, check Read-only, then click OK.

That's it! You can use your custom spray right now.

Note for Sven Co-op players

You have to enable sprays in game option. To enable sprays, open Options, go to Multiplayer tab, click Advanced..., go to Graphics: Custom player sprays, select Enabled. I recommend Large cache (16MB).

Note for pixel art sprays

You have to set gl_texturemode GL_NEAREST_MIPMAP_LINEAR from console to get shaper effect.
