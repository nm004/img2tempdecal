Img2tempdecal Web
====================

Image tempdecal.wad converter running in a web browser.

Build Requirements
----------------

 * [pnpm](https://pnpm.io/)
 * [wasm-pack](https://github.com/rustwasm/wasm-pack)

Build
----------------

```
# Development build
pnpm run dev

# Production build (output in 'dist')
pnpm run build

# Preview final output
pnpm exec vite preview
```
